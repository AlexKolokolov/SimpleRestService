REST сервис, при обращенни к которому по URI localhost:8080/hello/contacts
выдается JSON массив из 100 контактов из базы данных (первых по id).
При добавлении к URI параметра nameFilter=regex (regex - регулярное выражение),
массив будет содержать только те контакты из базы данных, в которых поле name не совпадает
с заданным регулярным выражением. Их количество также не будет превышать 100 штук.

В папке src/main/resources находится файл DBConnection.properties, в котором необходимо
перед созданием артефакта изменить значения полей host, port, user, password на соответствующие
Вашей базе данных PostgreSQL, в которой содержится таблица contacts, описанная в Вашем задании.

После создания артефакта (командой mvn clean package) программу запускать с
помощью start.bat, чтобы при запуске открылось окно консоли, которое можно будет
в последствии закрыть и таким образом остановить работу сервиса.
